# Laravel API Toolkit - V3 Documentation

Welcome to the Laravel API Toolkit documentation for **V3** — the latest version with AI-powered architectural rules, enhanced features, and modern PHP support.

The Laravel API Toolkit is a comprehensive suite of tools designed to help you create robust, high-performance APIs using Laravel. From standardized responses to advanced filtering — and now **AI architectural skills** that teach your coding assistant to follow your standards — everything you need to ship production-grade APIs.

## What's New in V3.1

- 🤖 **AI Architectural Skill** — 20 rule files + 5 workflows that teach your AI coding assistant your architecture. One command install for Claude Code, Cursor, GitHub Copilot, and Antigravity.

## Key Features

- 🤖 **[AI Skill](/v3/ai-skill.md)** — Deep Project Scan & Architectural Rules (new in v3.1)
- 📋 **[API Response](/v3/api-response.md)** — Standardized response formatting
- 🔍 **[Filters](/v3/filters.md)** — Advanced filtering, sorting, and search
- ⚙️ **[Actions](/v3/actions.md)** — Clean business logic with the command pattern
- 📖 **[Dynamic Pagination](/v3/dynamic-pagination.md)** — Flexible, client-driven pagination
- 🖼️ **[Media Helper](/v3/media-helper.md)** — Easy file upload and management
- 🏷️ **[Enums](/v3/enum.md)** — Type-safe enumerations
- 🚀 **[API Generator](/v3/api-generator.md)** — Scaffold API endpoints (for traditional workflows)

## Getting Started

1. Install the package: `composer require essa/api-tool-kit`
2. Install the **[AI Skill](/v3/ai-skill.md)**: `php artisan api-skill:install`
3. Explore the documentation using the **sidebar navigation**
4. Review the [Migration Guide](/v3/migration-guide.md) if upgrading from V2
